Python is a general-purpose interpreted, interactive, object-oriented, and high-level
programming language. It was created by Guido van Rossum during 1985- 1990. Like Perl,
Python source code is also available under the GNU General Public License (GPL). This tutorial
gives enough understanding on Python programming language.This tutorial is designed for software programmers who need to learn Python programming
language from scratch.You should have a basic understanding of Computer Programming terminologies. A basic
understanding of any of the programming languages is a plus.Python distribution is available for a wide variety of platforms. You need to download only the
binary code applicable for your platform and install Python.If the binary code for your platform is not available, you need a C compiler to compile the
source code manually. Compiling the source code offers more flexibility in terms of choice of
features that you require in your installation.
